# @fluentui-contrib/cap-foundations-core

Core foundation layer for the Cap design system: static design tokens, color utilities, theme generation pipeline, and zero-flash runtime bootstrap.

## Installation

```bash
yarn add @fluentui-contrib/cap-foundations-core
```

## Packages in this workspace

This package is part of the [cap-foundations](../) umbrella.

## Building

```bash
nx run cap-foundations-core:build
nx run cap-foundations-core:test
nx run cap-foundations-core:type-check
```
