# @fluentui-contrib/cap-foundations-react

React theme provider and components built on [cap-foundations](../README.md) design tokens.

## Installation

```sh
yarn add @fluentui-contrib/cap-foundations-react @fluentui-contrib/cap-foundations-core
```

## Usage

### ThemeProvider

Wrap your application with `ThemeProvider` to bootstrap the theme runtime and subscribe React to theme changes.

```tsx
import { ThemeProvider } from '@fluentui-contrib/cap-foundations-react';

function App() {
  return (
    <ThemeProvider>
      <YourApp />
    </ThemeProvider>
  );
}
```

### useTheme

Read and control the active theme from any component inside `ThemeProvider`.

```tsx
import { useTheme } from '@fluentui-contrib/cap-foundations-react';

function ThemeSwitcher() {
  const { theme, mode, resolvedMode, toggleMode, setMode } = useTheme();

  return <button onClick={toggleMode}>{resolvedMode === 'light' ? '🌙 Dark' : '☀️ Light'}</button>;
}
```

### ThemeProvider Props

| Prop           | Type                          | Description                                                             |
| -------------- | ----------------------------- | ----------------------------------------------------------------------- |
| `children`     | `ReactNode`                   | Required. Application tree.                                             |
| `defaultTheme` | `string`                      | Initial theme name (default: `'default'`). Persisted to `localStorage`. |
| `defaultMode`  | `'light' \| 'dark' \| 'auto'` | Initial mode (default: `'auto'`).                                       |

### useTheme Return Value

| Property       | Type                          | Description                          |
| -------------- | ----------------------------- | ------------------------------------ |
| `theme`        | `string`                      | Active theme name                    |
| `mode`         | `'light' \| 'dark' \| 'auto'` | Current mode setting                 |
| `resolvedMode` | `'light' \| 'dark'`           | Resolved mode (auto → light or dark) |
| `setTheme`     | `(theme, mode?) => void`      | Change theme and/or mode             |
| `setMode`      | `(mode) => void`              | Change mode only                     |
| `toggleMode`   | `() => void`                  | Cycle: light → dark → auto → light   |

## Theme CSS

Theme CSS files are generated by `cap-foundations-core` and must be available at runtime. Load them in your entry point:

```html
<!-- Option 1: static link in HTML -->
<link rel="stylesheet" href="/themes/default-light.css" />

<!-- Option 2: bootstrap script (zero-flash) -->
<script src="/node_modules/@fluentui-contrib/cap-foundations-core/dist/bootstrap.js"></script>
```

Or import in JS (may cause a brief flash on first load):

```ts
import '@fluentui-contrib/cap-foundations-core/dist/themes/default-light.css';
```

## Storybook

```sh
# Build theme CSS first
nx run cap-foundations-core:build-themes

# Start Storybook
nx run cap-foundations-react:storybook
```
